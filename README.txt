###Recipe Box

### An Application that allows users to save lists of ingredients and maket them into a recipe list 08/28/2015

### By Travis Crampton & Christoff Perlinger

##Description

#This application allows users to save lists of ingredients and combine said ingredients into a a recipe list, 
# for a combination of ingredients to make up a recipe

## Setup

* Run postgres
* in terminal rake db:create
* in terminal rake db:migrate

*this should set up the database, dont forget to bundle all of the gems


# languages used

Ruby sql
used program active record
